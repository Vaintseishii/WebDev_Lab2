# GearHub Mini E-Commerce
GearHub is a single-page React app for browsing tech accessories, filtering products, managing a cart, and processing a simulated checkout.

## Members
- Xanth Reign Palmes
- Joseph Brian Azarraga (Vigger)

## Details
- NavBar.tsx - Displays the app title and cart icon with the item-count badge.
- ProductGrid.tsx - Arranges and renders the list of products in a grid.
- ProductCard.tsx - Displays the details of a single product and provides the "Add to Cart" action.
- FilterBar.tsx - Contains the search, category, price, and sort controls for filtering products.
- CartDrawer.tsx - Displays the slide-out shopping cart, totals, and checkout section.
- CartItem.tsx - Represents one item in the shopping cart, including quantity controls and remove functionality.
